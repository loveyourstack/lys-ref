# lys-ref

Reference implementation of LoveYourStack repos:

* [lys](https://github.com/loveyourstack/lys) - main backend library
* [connectors](https://github.com/loveyourstack/connectors) - external API connectors
* [lys-ref](https://github.com/loveyourstack/lys-vue) - Vue UI components and Typescript functions. Import via [npm](https://www.npmjs.com/package/lys-vue)

This repo is deployed to [ref.loveyourstack.dev](https://ref.loveyourstack.dev/home). Please contact me if you need login credentials.
